# latest-version-cli

> Get the latest version of an npm package


## Install

```
$ npm install --global latest-version-cli
```


## Usage

```
$ latest-version --help

  Usage
    $ latest-version <package-name>

  Example
    $ latest-version ava
    0.18.0
```
